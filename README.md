# Eleventy + Nunjucks Templating Demo

A small multi-page site generated with **Eleventy (11ty)** using **Nunjucks layouts + partials**.

## What this demonstrates
- **Templating** with a base layout and reusable partials (head, header, footer)
- **Multi-page generation**: Home, About, Contact
- **Static assets passthrough**: CSS/JS copied from `src/assets` into the build output

## Run locally
```bash
npm install
npm run dev
